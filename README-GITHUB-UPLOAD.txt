GitHub Pages upload instructions

1. Extract this ZIP file completely before uploading.
2. Upload every extracted file and folder to the repository root.
3. Keep the assets folder in the same location as index.html.
4. Do not upload the ZIP file itself as the website source.
5. Confirm that assets/fortescue-zero-logo.png and the JPG files are visible in the repository before publishing.

The homepage must remain at the repository root as index.html.
